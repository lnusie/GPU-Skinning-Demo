# GPU-Skinning-Demo
Unity 多人同屏技术测试Demo

实现内容
1.将顶点动画烘焙为动画贴图
2.通过动画贴图进行顶点动画
3.替换SkinMeshRenderer为MeshRenderer以应用GPU Instancing技术
4.实现全新的一套Animator以实现动画切换、动画融合等功能。
